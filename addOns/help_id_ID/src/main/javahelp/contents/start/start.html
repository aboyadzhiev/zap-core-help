The quickest way to get going with ZAP is to use the Quick Start add-on, which is installed by default.
This allows you to enter a URL which ZAP will first spider and then active scan.
For a more in depth test you should explore your application using your browser or automated regression tests while proxying through ZAP.

At its heart ZAP is a manipulator-in-the-middle proxy.
To get the most out of ZAP you need to configure your browser or functional tests to connect to the web application you wish to test through ZAP.
If required you can also configure ZAP to connect through another proxy - this is often necessary in a corporate environment.

Anda dapat meluncurkan peramban dengan cepat dan mudah yang sebelumnya telah dikonfigurasikan ke proxy melewati ZAP melalui tab Mulai Cepat. Ini dapat meluncurkan salah satu peramban yang paling umum yang telah anda pasang dengan profil baru.
Jika anda ingin menggunakan salah satu peramban dengan profil yang telah ada, sebagai contoh dengan add-ons peramban lain yang terpasang, kemudian anda perlu mengkonfigurasi peramban secara manual ke proxy melalui ZAP.
Jika anda mengetahui bagaimana mengatur proxy pada peramban web anda, silahkan lanjutkan dan mencobanya!
Jika anda tidak yakin maka lihatlah pada bagian Configure proxies.

Setelah anda telah mengkonfigurasi ZAP sebagai proxy pramban anda kemudian coba untuk menghubungkan pada aplikasi web yang akan di tes.
Jika anda tidak dapat terhubung kemudian cek kembali pengaturan proxy anda. Anda perlu memeriksa pengaturan proxy peramban anda, dan juga pengaturan proxy ZAP.
Hal ini juga layak dicermati bahwa aplikasi yang ingin anda uji sedang berjalan!

Ketika anda berhasil terhubung ke aplikasi anda melalui peramban anda kemudian liat ZAP kembali. Sekarang anda akan melihat satu atau lebih baris dalam Situs dan Riwayat tab.
Jika demikian kita berada dalam bisnis. Jika tidak maka anda perlu untuk mengecek pengaturan proxy anda kembali.

The next thing to do is to start a basic penetration test.
