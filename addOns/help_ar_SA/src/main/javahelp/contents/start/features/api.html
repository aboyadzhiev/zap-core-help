API

انطلق تنص واجهة برمجة التطبيقات (API) التي تتيح لك التفاعل مع زاب برمجياً.

API متاحة في تنسيقات JSON و HTML و XML.
صفحة ويب بسيطة واجهة المستخدم الذي يسمح لك لاستكشاف واستخدام API متوفرة عبر عنوان URL http://zap/ عندما كنت proxying عبر انطلق، أو عن طريق المضيف ومنفذ الاستماع انطلق، على سبيل المثال http://localhost:8080/.

بشكل افتراضي فقط انطلق قيد التشغيل على الجهاز قادرة على الوصول إلى API. يمكنك السماح الأجهزة الأخرى التي هي قادرة على استخدام انطلق كبديل الوصول إلى API. API تم تكوينه باستخدام خيارات API الشاشة.

The API provides access to most of the core ZAP features such as the active scanner.
Future versions of ZAP will increase the functionality available via the APi.
