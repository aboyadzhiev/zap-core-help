Pilihan periksa layar pembaruan

Layar ini memungkinkan anda untuk mengkonfigurasiapaqkah ZAP melihat untuk memeriksa untuk meleihat apakah anda sedang menjalankan versi baru saat dimulai, dan jika demikian apa yang anda lakukan.

Periksa pembaruan saat distartup

Jika dipilih maka ZAP secara otomatis akan memeriksa sebagai pembaruan terhadap keseluruhan aplikasi dan semua pengaya ketika itu dimulai.
Sangat disarankan supaya anda memeriksa kotak ini.
Jika karena alasan apapun anda memilih untuk tidak maka anda harus sering memeriksa pembaruan secara manual.
ZAP hanya akan membuat sebuah permintaan, dan satu-satunya informasi yang disertakan akan menjadi versi terbaru yang akan anda gunakan.

Automatically download new ZAP releases.

Jika dipilih maka ZAP akan secara otomatis mengunduh rilis ZAP baru ketika itu tersedia dan menyarakan anda untuk memasangnya.

Automatically install updates to the add-ons you have installed.

Jika dipilih kemudian ZAP akan secara otomatis mengunduh dan memasang pembaruan untuk pegaya yang telah anda pasang.

Automatically install updates to the scan rules you have installed.

If selected then ZAP will automatically download and install any updates to the scan rules you have installed.
Ini merupakan subset dari set lengkap untuk pengaya yang terpasang.

Report new Release status add-ons.

If selected then ZAP will inform you if and when any add-ons are promoted to 'release' status.
These add-ons will have been thoroughly tested and reviewed, and you can be sure they are of the highest status.

Report new Beta status add-ons.

If selected then ZAP will inform you if and when any add-ons are promoted to 'beta' status.
These add-ons will have been tested and reviewed, and are considered to be of a reasonable status and mostly fit for purpose.
Namun mereka mungkin tidak lengkap atau membutuhlan lebih lanjut pengujian.
Beberapa pengaya yang disertakan dengan ZAP secara bawaan masih berada pada level beta.

Report new Alpha status add-ons.

If selected then ZAP will inform you if and when any new add-ons are added at 'alpha' status.
Pengaya ini akan ditinjau namun biasanya pada tahapan awal dari pengembangan.
Mereka mungkin tidak lengkap, mengandung masalah signifikan atau menyebabkan masalah stabilitas.

Tambahkan direktori

ZAP akan memuat semua pengaya dari direktori plugin bawaan.
Anda juga dapat menambahkan direktori add-on tambahkan sebanyak yang anda butuhkan dengan menambahkan mereka di sini.
Ini bisa sangat berguna ketika menjalankan beberapa ZAP misalnya pada satu mesin, sebagai contoh pada lingkungan CI.
Kemudian anda bisa mengunduh pengaya baru atau yang pernah diperbarui dan segera membagikannya di antara beberapa contoh.
Pegaya yang diunduh kedalam direktori bersama hanya akan ditambahkan kekasus ZAP baru, tidak untuk orqang orang yang sudah berjalan, dengan pengecualian contoh yang benar benar diunduh.

Unduh direktori

Ini merupakan direktori ZAP yang akan mengunduh pengaya baru atau yang telah diperbarui.
Secara deafult, itu akan menjadi direktori lokal, tetapi anda juga dapat memilioh salah satu direktori bersama sebagai contoh ZAP ini dapat menulis untk itu.
ZAP ini hanya kaan bergerqak pegaya ke direktori bersama setelah mereka sepenuhnya diunduh, sehingga kasus ZAP lain tidak mencoba untuk memuat add-on tidak lengkap.
