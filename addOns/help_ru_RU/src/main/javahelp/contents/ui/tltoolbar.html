Панель инструментов верхнего уровня

Эта панель инструментов содержит набор элементов управления для часто используемых функций.

Раскрывающийся список режимов

Это позволяет изменить текущий режим.

  Новый сеанс

Эта кнопка аналогична пункту меню Файл «Новый сеанс».

  Открыть сеанс

Эта кнопка аналогична пункту меню Файл «Открыть сеанс».

  Сохранить сеанс...

Эта кнопка аналогична пункту меню Файл «Сохранить сеанс...».

  Снимок сеанса

Эта кнопка аналогична пункту меню Файл «Снимок сеанса».

  Свойства сеанса...

Эта кнопка аналогична пункту меню Файл 'Свойства...'.

  Параметры...

Эта кнопка аналогична пункту меню Сервис 'Параметры...'.

  Динамическое переключение внешнего вида

Это позволяет вам динамически переключать внешний вид, который определяет внешний вид пользовательского интерфейса ZAP Desktop и включает поддержку «темных режимов».

  Показать все вкладки

Эта кнопка показывает все скрытые вкладки.

  Скрыть незакрепленные вкладки

Эта кнопка скрывает все незакрепленные вкладки. Вкладки можно закреплять и откреплять с помощью маленького значка «булавки», который отображается при выборе вкладки.

  Show Tab Names and Icons

This button toggles the displaying of the tab names.

  Развернуть вкладку "Сайты"

Это изменяет отображение таким образом, что окно «дерево», содержащее вкладку «Сайты», расширяется на всю длину левая сторона.
Это уменьшит количество места, доступного для информационного окна.

  Развернуть информационные вкладки

Это изменяет отображение, так что окно «информации» расширяется на всю длину нижней части.
Это уменьшит количество места, доступного для окна «дерева».

  Полный макет

Это изменяет отображение так, что выбранная вкладка занимает весь экран.
Это полезно при использовании ZAP на маленьких экранах.

  Request and Response Tabs Side by Side

This changes the display so that the request and response tabs are side by side.
This increases the information that can be displayed but means you cannot see both the request and response at the same time.

  Request and Response Panels Side by Side in the Same Tab

This changes the display so that the request and response panels are shown side by side in the same tab.
This decreases the information that can be displayed but means you can see both the request and response at the same time.

  Request Shown above Response

This changes the display so that the request panel is shown above the response panel.
This decreases the information that can be displayed but means you can see both the request and response at the same time.

  Request and Response Panels Side by Side

This changes the display so that the request panel is shown to the left of the response panel.
This decreases the information that can be displayed but means you can see both the request and response at the same time.

 /    Set / Unset Break on All Requests and Responses

This sets and unsets a 'global' breakpoint that will trap and display the next request or response in the Break tab.
You can then change any part of the request or response that you want to and send it to the target application by pressing either of the 'Step' or 'Continue' buttons.
Alternatively you can press the 'Drop' button to dispose of the request or response.
You can switch between a single 'combined' break button and separates ones for requests and responses via the Options breakpoints screen

 /    Set / Unset Break on All Requests

This sets and unsets a 'global' breakpoint that will trap and display the next request in the Break tab.
You can then change any part of the request that you want to and send it to the target application by pressing either of the 'Step' or 'Continue' buttons.
Alternatively you can press the 'Drop' button to dispose of the request.
You can switch between a single 'combined' break button and separates ones for requests and responses via the Options breakpoints screen

 /    Set / Unset break on all responses

This sets and unsets a 'global' breakpoint that will trap and display the next response in the Break tab.
You can then change any part of the response that you want to and send it to your browser by pressing either of the 'Step' or 'Continue' buttons.
Alternatively you can press the 'Drop' button to dispose of the request.
You can switch between a single 'combined' break button and separates ones for requests and responses via the Options breakpoints screen

  Step

This allows the trapped request or response to continue to the application or your browser with any changes that you have made to it.
The 'global' breakpoint will remain set so that the next request or response will also be caught.
This button is only enabled when a request or response is trapped.

  Продолжить

"Глобальная" точка останова не будет установлена, поэтому последующие запросы и ответы больше не будут перехватываться ZAP, если только вы не установите точки останова на определенных URL-адресах.
Эта кнопка активна только в том случае, если запрос или ответ перехвачены.

  Бросить

Это отбрасывает захваченный запрос или ответ, чтобы он не был передан приложению. или в вашем браузере.
Эта кнопка активна только в том случае, если запрос или ответ перехвачены.

  Добавить пользовательскую точку останова HTTP

Откроется диалоговое окно Добавить точку останова, в котором можно указать критерии для точка останова.

  Диспетчер политики сканирования...

Отображает диалоговое окно Диспетчер политик сканирования, в котором можно настроить политики сканирования.

  Управление надстройками

Появится диалоговое окно Управление надстройками, которое позволяет вам находите, устанавливайте и обновляйте надстройки из интернет-магазина.
Он также позволяет удалять надстройки.

 /    Принудительный режим пользователя вкл./выкл.

Это включает и выключает принудительный пользовательский режим.
Кнопка активна только в том случае, если вы определили принудительного пользователя хотя бы для одного контекст, что можно сделать с помощью Диалоговое окно Контексты сеанса.

Обратите внимание, что надстройки могут добавлять дополнительные кнопки.
