接口

ZAP提供了一个应用程序编程接口(API)，它允许您以编程方式与ZAP交互。

API以JSON，HTML和XML格式提供。
一个简单的网络用户界面，允许您探索和使用该API，只需通过URL http://zap/ 当您通过ZAP进行代理，或通过ZAP正在监听的主机和端口，例如 http://localhost:8080/.

默认情况下，只有运行的机器ZAP才能访问API。 您可以允许其他的机器，能够采用ZAP公司作为代理，访问API。 该API使用API选项屏幕进行配置.

The API provides access to most of the core ZAP features such as the active scanner.
Future versions of ZAP will increase the functionality available via the APi.
