Accessoires

Les accessoires ajoutent des fonctionnalités supplémentaires à ZAP.
Ils ont un accès complet à toutes la mécanique interne de ZAP et peuvent donc fournir de nouvelles fonctionnalités très puissantes.
Vous pouvez installer dynamiquement les accessoires à partir de la foire aux modules en ligne via la boîte de dialogue Gérer les accessoires complémentaires.
Généralement, vous pouvez ajouter et supprimer des accessoire depuis l'interface utilisateur de ZAP sans avoir à redémarrer l'application.

Add-ons can provide context sensitive help which is dynamically integrated with the help file.
If you are reading this page via a ZAP help file (as opposed to reading it online) then any help pages associated with the add-ons you have installed will be available under here.

Les accessoires reçoivent un statut parmi les suivants :
     Releasewhich indicates they are of high status and fit for purpose
     Bêtawhich indicates they are of reasonable status but may be incomplete or need further testing
     Alphaqui indique qu'ils sont à un stade précoce du développement

Location of add-ons available to ZAP

To make an add-on available to ZAP it must be in one of the following locations:
  • plugin directory located in the ZAP's installation folder;
  • plugin directory located in the ZAP's user folder.
