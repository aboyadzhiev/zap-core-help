Modus

ZAP memiliki 'mode' yang dapat:

  • Safe - no potentially dangerous operations permitted.
  • Protected - you can only perform (potentially) dangerous actions on URLs in the scope.
  • Standard - does not restrict anything.
  • ATTACK - new nodes that are in scope are actively scanned as soon as they are discovered.
Disarankan bahwa anda menggunakan mode Perlindungan untuk memastikan bahwa anda hanya menyerang situs yang anda maksud.

Mode dapat diubah melalui toolbar (atau ZAP API) dan bertahan diantara sesi.

Contoh dari hal-hal yang tidak akan mungkin dilakukan dalam mode Aman atau dalam mode Perlindungan ketika tidak bertindak berdasarkan URL dalam Lingkupan:
  • Spindering
  • Pemindaian aktif
  • Fuzzing
  • Penjalajahan paksa
  • Pemecahan (mencegat)
  • Mengirim ulang permintaan
Anda dapat menentukan Kebijakan Pindai untuk digunakan sebagai mode serangan Pilihan aktif Scan layar.
