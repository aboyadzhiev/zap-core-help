Iniciando

The quickest way to get going with ZAP is to use the Quick Start add-on, which is installed by default.
This allows you to enter a URL which ZAP will first spider and then active scan.
For a more in depth test you should explore your application using your browser or automated regression tests while proxying through ZAP.

At its heart ZAP is a manipulator-in-the-middle proxy.
To get the most out of ZAP you need to configure your browser or functional tests to connect to the web application you wish to test through ZAP.
If required you can also configure ZAP to connect through another proxy - this is often necessary in a corporate environment.

Você pode, de uma maneira rápida e fácil, iniciar navegadores com o proxy pré-configurado, através da aba Inicio rápido. Isto pode iniciar os navegadores mais comuns, que você tem instalado, com novos perfis.
Se você quiser usar um dos seus browsers, com um perfil já existente, com outros complementos instalados, então você precisará configurar o seu navegador para usar o ZAP como proxy.
Se você sabe como configurar proxies no seu navegador, pode seguir adiante e configurar!
Caso você não esteja seguro, verifique a sessão Configurando proxies.

Uma vez que você tenha configurado o ZAP como proxy do seu navegador, em seguida, tente se conectar ao aplicativo da web que você estará testando.
Se você não pode se conectar a ele então verifique suas configurações de proxy novamente. Você precisará verificar as configurações de proxy do seu browser e também as configurações de proxy do ZAP.
Também vale a pena verificar se o aplicativo que você está tentando testar está ativo!

Quando você conectar ao seu aplicativo através do seu navegador, dê uma olhada no ZAP novamente. Você então deve ver uma ou mais linhas nas guias Sites e histórico.
Se isso aconteceu, está tudo funcionando. Se não, então você precisará verificar as configurações de proxy de novo.

The next thing to do is to start a basic penetration test.
