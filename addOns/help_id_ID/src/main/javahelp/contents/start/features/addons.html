Pengaya

Add-on menambahkan fungsionalitas tambahan ke ZAP.
Mereka memiliki akses penuh ke semua internal ZAP, sehingga bisamemberikan fitur baru yang sangat kuat.
Anda dapat memasang pengaya secara dinamis dari Pasar Pengaya secara online melalui dialog Kelola Pengaya.
Anda dapat secara khusus menambahkan dan menghapus pengaya ke dan dari UI ZAP tanpa harus memulai ulang.

Add-ons can provide context sensitive help which is dynamically integrated with the help file.
If you are reading this page via a ZAP help file (as opposed to reading it online) then any help pages associated with the add-ons you have installed will be available under here.

Add-on diberi status yang mungkin salah satu dari:
     Melepaskanwhich indicates they are of high status and fit for purpose
     Betawhich indicates they are of reasonable status but may be incomplete or need further testing
     Alfayang menunjukkan bahwa mereka berada pada tahap awal dari pengembangan

Lokasi pengaya tersedia untuk ZAP

Untuk menambahkannya ke ZAP, Anda harus berada di salah satu lokasi berikut:
  • plugin yang terletak di berkas instalasi ZAP;
  • plugin yang terletak di berkas pengguna ZAP.
