Пассивное сканирование

ZAP по умолчанию пассивно сканирует все HTTP-сообщения (запросы и ответы), отправляемые тестируемому веб-приложению. <0 /> Пассивное сканирование никоим образом не меняет ни запросы, ни ответы, поэтому его можно безопасно использовать. <0 /> Сканирование выполняется в фоновом потоке, чтобы убедиться, что это не замедляет исследование. приложения.

(Основное) поведение пассивного сканера можно настроить с помощью Параметры экрана пассивного сканера .

Пассивное сканирование также можно использовать для автоматического добавления тегов и создание предупреждений о потенциальных проблемах.
По умолчанию предоставляется набор правил для автоматической пометки. Их можно изменить, удалить или добавлено на экране Параметры пассивного сканирования .

The alerts raised by passive scan rules can be configured using the Options Passive Scan Rules screen.
