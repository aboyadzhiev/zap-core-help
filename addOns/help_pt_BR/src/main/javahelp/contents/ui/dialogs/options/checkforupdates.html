Tela das Opções para verificar Atualizações

Esta tela permite que você configure se o ZAP verificará se você está executando a versão mais recente quando começa, e em caso contrário o que deve fazer.

Verificar atualizações ao iniciar

Se selecionado, então o ZAP irá automaticamente verificar se há atualizações para o aplicativo inteiro e para todos os add-ons quando iniciar.
É altamente recomendável que você ligue esta caixa.
Se por algum motivo que você escolher não, deverá verificar manualmente e com frequência as atualizações.
ZAP vai fazer apenas um pedido de atualização e a única informação incluída será a versão atual em que ele está.

Automatically download new ZAP releases.

Se selecionado, então o ZAP irá automaticamente baixar novos releases ZAP quando eles estiverem disponíveis e pedir para você instalá-los.

Automatically install updates to the add-ons you have installed.

Se selecionado, o ZAP fará automaticamente o download e instalará quaisquer updates nos add-ons que você tiver no ZAP.

Automatically install updates to the scan rules you have installed.

If selected then ZAP will automatically download and install any updates to the scan rules you have installed.
Estes são um subset do conjunto completo de Add-ons instalados.

Report new Release status add-ons.

If selected then ZAP will inform you if and when any add-ons are promoted to 'release' status.
These add-ons will have been thoroughly tested and reviewed, and you can be sure they are of the highest status.

Report new Beta status add-ons.

If selected then ZAP will inform you if and when any add-ons are promoted to 'beta' status.
These add-ons will have been tested and reviewed, and are considered to be of a reasonable status and mostly fit for purpose.
No entanto, eles podem estar incompletos ou precisar de mais testes.
Alguns dos add-ons incluídos no ZAP por padrão estão ainda no nível beta.

Report new Alpha status add-ons.

If selected then ZAP will inform you if and when any new add-ons are added at 'alpha' status.
Estescomplememntos estarão revisados, mas geralmente estão na fase inicial de desenvolvimento.
Podem estar incompletos, conter problemas relevantes ou causar problemas de estabilidade.

Diretórios de complementos

ZAP vai carregar todos os complementos do diretório padrão plugin.
Você também pode adicionar tantos diretórios de complementos quanto precisar, adicionando-os aqui.
Isso pode ser muito útil quando várias instâncias do ZAP estiverem sendo executadas em uma máquina, por exemplo um ambiente de CI.
Você pode então apenas baixar complementos novos ou atualizados e imediatamente compartilhá-los entre as várias instâncias.
Complementos baixados em diretórios compartilhados serão adicionados apenas para novas instâncias do ZAP, e não para aqueles que já estão em execução, com exceção da instância que baixou aquele complemento.

Diretório de Download

Este é o diretório onde o Zap irá baixar os complementos novos ou atualizados.
Por padrão este será o diretório local, mas você também pode escolher um dos diretórios compartilhados, contanto que esta instância do ZAP possa escrever nele.
O ZAP só moverá Complementos para um diretório compartilhado depois que eles estiverem completamente baixados, para que outras instâncias ZAP não tentem carregar um complemento incompleto.
