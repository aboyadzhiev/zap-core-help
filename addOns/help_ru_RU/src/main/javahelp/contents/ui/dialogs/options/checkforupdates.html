Экран «Проверить наличие обновлений»

Этот экран позволяет вам настроить, будет ли ZAP проверять, работаете ли вы последняя версия, когда она запускается, и если да, то что она делает.

Проверять наличие обновлений при запуске

Если выбрано, ZAP будет автоматически проверять наличие обновлений для всего приложения и всех надстроек при запуске.
Настоятельно рекомендуется установить этот флажок.
Если по какой-либо причине вы решите этого не делать, вам следует часто вручную проверять наличие обновлений.
ZAP сделает только один запрос, и единственной включенной информацией будет текущая версия, на которой вы работаете.

Automatically download new ZAP releases.

Если выбрано, ZAP будет автоматически загружать новые выпуски ZAP, когда они будут доступны, и предлагать их установить.

Automatically install updates to the add-ons you have installed.

Если выбрано, ZAP автоматически загрузит и установит все обновления для установленных вами надстроек.

Automatically install updates to the scan rules you have installed.

If selected then ZAP will automatically download and install any updates to the scan rules you have installed.
Это подмножество полного набора установленных надстроек.

Report new Release status add-ons.

If selected then ZAP will inform you if and when any add-ons are promoted to 'release' status.
These add-ons will have been thoroughly tested and reviewed, and you can be sure they are of the highest status.

Report new Beta status add-ons.

If selected then ZAP will inform you if and when any add-ons are promoted to 'beta' status.
These add-ons will have been tested and reviewed, and are considered to be of a reasonable status and mostly fit for purpose.
Однако они могут быть неполными или нуждаться в дальнейшем тестировании.
Некоторые надстройки, включенные в ZAP по умолчанию, все еще находятся на уровне бета-версии.

Report new Alpha status add-ons.

If selected then ZAP will inform you if and when any new add-ons are added at 'alpha' status.
Эти надстройки будут проверены, но обычно они находятся на ранней стадии разработки.
Они могут быть неполными, содержать серьезные проблемы или вызывать проблемы со стабильностью.

Каталоги дополнений

ZAP загрузит все дополнения из по умолчанию плагин каталогов.
Вы также можете добавить столько дополнительных каталогов, сколько вам нужно, добавив их здесь.
Это может быть очень полезно при запуске нескольких экземпляров ZAP на одном компьютере, например, в среде CI.
Затем вы можете просто загрузить новые или обновленные надстройки один раз и сразу же поделиться ими между несколькими экземплярами.
Надстройки, загруженные в общие каталоги, будут добавляться только в новые экземпляры ZAP, а не в уже запущенные. за исключением экземпляра, который фактически загрузил надстройку.

Скачать каталог

Это каталог, в который ZAP будет загружать новые или обновленные дополнения.
По умолчанию это будет локальный каталог, но вы также можете выбрать один из общих каталогов, если этот ZAP экземпляр может писать в него.
ZAP будет перемещать надстройки в общий каталог только после их полной загрузки, чтобы другие экземпляры ZAP не пытайтесь загрузить неполное дополнение.
