режимы

ZAP имеет «режим», который может быть:

  • Safe - no potentially dangerous operations permitted.
  • Protected - you can only perform (potentially) dangerous actions on URLs in the scope.
  • Standard - does not restrict anything.
  • ATTACK - new nodes that are in scope are actively scanned as soon as they are discovered.
Рекомендуется использовать защищенный режим, чтобы гарантировать, что вы атакуете только те сайты, которые намереваетесь.

Режим можно изменить с помощью панель инструментов (или ZAP API) и сохраняется между сеансами.

Примеры того, что будет невозможно ни в безопасном, ни в защищенном режиме, если не действовать на URL-адреса в области действия:
  • Spidering ( просмотре)
  • Активное сканирование
  • Фаззинг
  • Принудительный просмотр
  • Взлом (перехват)
  • Повторная отправка запросов
Вы можете определить Политика сканирования для использования в режиме атаки Параметры экрана активного сканирования .
